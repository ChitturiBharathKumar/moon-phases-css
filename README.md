# 🌙 CSS Moon Phases

A simple HTML and CSS project that demonstrates different moon phases using only CSS `box-shadow` and `border-radius` properties.

## 📸 Preview

This project displays 16 moon phase variations created without using images or JavaScript.

## ✨ Features

- Pure HTML & CSS
- Uses `box-shadow` to create moon phase effects
- Responsive and lightweight
- Beginner-friendly project
- No external libraries required

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
css-moon-phases/
│── index.html
│── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/css-moon-phases.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 📚 What You'll Learn

- CSS `box-shadow`
- `border-radius`
- CSS positioning
- Creating shapes with CSS
- Basic HTML structure
